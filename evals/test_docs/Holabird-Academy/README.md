**Project Number:** 2024-0312
**Owner:** Baltimore City Public Schools

## Project Description

New construction of a 95,000 SF elementary/middle school facility serving grades K-8 for Baltimore City Public Schools. The project includes two academic buildings (Building A — classroom wing, Building B — gymnasium/cafeteria), site improvements including parking, playgrounds, and athletic fields, and full MEP systems.

**Contract Sum:** $27,600,000
**Notice to Proceed:** May 16, 2025
**Substantial Completion:** March 18, 2026

## Folder Structure

```
01 - Drawings/          ← Drawing PDFs organized by discipline
02 - Specifications/    ← Project manual / spec sections
03 - Contract Documents/ ← Agreement, general conditions, insurance
04 - Addenda/           ← Issued addenda
05 - RFIs/              ← Requests for Information + RFI Log
06 - Submittals/        ← Submittal packages by CSI section + Submittal Log
07 - Meeting Minutes/   ← OAC progress meeting minutes
08 - Daily Reports/     ← Daily construction logs
09 - Change Orders/     ← COs, PCOs, and Change Order Log
10 - Pay Applications/  ← Monthly pay applications (AIA G702/G703)
11 - Photos/            ← Progress photos organized by month
12 - Schedule/          ← Master schedule and lookaheads
13 - Correspondence/    ← Letters, emails, and correspondence log
14 - Punch List/        ← Punch list (template — not yet at this phase)
15 - Closeout/          ← Closeout documents (not yet at this phase)
```
