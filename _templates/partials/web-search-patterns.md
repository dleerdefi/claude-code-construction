### Web Search Best Practices

When using WebSearch and WebFetch for construction research:
- Prefer .gov and official standards body domains
- Always verify publication date — construction codes change on 3-year cycles
- Cross-reference web findings against project spec requirements
- Never cite a web source without verifying against the applicable code edition
- When WebFetch returns a PDF, extract only the relevant sections
- Do NOT trust blog posts, contractor marketing sites, or undated secondary sources
